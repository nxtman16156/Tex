Installing for Windows

You use the Basic MiKTeX Installer to install MiKTeX on your Windows computer. You can download the installer from the MiKTeX download page.

Please read the installation tutorial, for a step-by-step guide.

Removing MiKTeX

MiKTeX can be removed from your computer in the following way:

  1. Open Control Panel.

  2. Double-click Add or Remove Programs.

  3. Click MiKTeX 2.9.

  4. Click Change/Remove.
